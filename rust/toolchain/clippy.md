# Description
``clippy`` is a **linter** for Rust. 
<br>
The **linter** (or **lint**), is a **static code analysis tool**. 

<br>

# ``clippy`` examples
|Example|Explanation|
|:------|:----------|
|rustup **component add** ``clippy``|Adds ``clippy`` **component** to the **current toolchain**.|
|cargo ``clippy``|Runs ``clippy`` for the **current crate**.|
